# StudyGenie

## Problem Statement
Students struggle with managing study schedules and getting instant doubt resolution, leading to inefficient learning.

## Solution
StudyGenie is an AI-powered web application that generates personalized study plans and provides instant AI-based doubt solving.

## Features
- Personalized study planner
- AI-based doubt solver
- Simple and user-friendly interface

## Tech Stack
- HTML
- CSS
- JavaScript
- AI (API / prompt-based)

## Future Scope
- User authentication
- Progress tracking
- Mobile app version
